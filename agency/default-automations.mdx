## 5 Automations
- Missed Call Textback
- Review Requests
- Estimate Follow-up
- New Lead Notification
- Appointment Reminder

## Setup
1. Agency Settings > Default Automations
2. Toggle on/off
3. Save Defaults

Apply to All Existing Clients button pushes defaults to every client.
